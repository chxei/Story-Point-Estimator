# Tech Stack & Coding Standards

- **Core Stack:** This project purely uses Vanilla HTML, CSS, and JavaScript.
- **No Frameworks:** Do NOT introduce build tools (like Webpack, Vite, or npm scripts for compilation) or UI frameworks (like React, Vue, Svelte, or Tailwind CSS) unless explicitly requested.
- **File Structure:** 
  - Keep all markup in `index.html`.
  - Keep all styles in `style.css`.
  - Keep all interactivity and calculator logic in `script.js`.
- **Code Style:** Use modern ES6+ features (const/let, arrow functions, DOMContentLoaded). Keep logic separated into clear, readable functions.
